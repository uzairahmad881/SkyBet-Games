This project was developed as part of a university event focused on innovation, creativity, and practical application of web technologies. Our goal was to design and implement an interactive, casino-style web application that demonstrates real-time game logic and engaging user experience.

The project includes multiple mini-games such as an Aviator-style crash game, a card-based reward system, and a random spin game. Each component was carefully designed to simulate real gaming mechanics while maintaining a smooth and responsive interface.

Throughout the development process, we focused on front-end technologies including HTML, CSS, and JavaScript, ensuring clean UI design, controlled game logic, and seamless animations. This project highlights our ability to build interactive web applications, manage game flow logic, and deliver an engaging digital experience.

Overall, this project reflects our practical skills, creativity, and teamwork demonstrated during the university event.
